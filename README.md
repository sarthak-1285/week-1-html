## ✅ Concepts Practiced

- Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- Proper heading hierarchy (`h1` → `h2`)
- Accessible form markup (`<label for="">`, `<fieldset>`, `<legend>`)
- Multiple input types (`text`, `email`, `number`, `textarea`, `select`)
- HTML validation attributes (`required`, `placeholder`)
- Git feature branching and commit workflow
